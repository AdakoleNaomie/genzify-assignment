A FullStack Engineer

A FullStack Engineer is an engineer that utilizes both frontend
and backend tools to build websites.
The front end includes everything that a user sees and interact
with when using a website.
The backend is everything behind the scenes for example,
the functionality and the complex parts of the website.
Frontend and BackEnd engineers have different roles in building a website.
However, A FullStack Engineer is a Master of both.
