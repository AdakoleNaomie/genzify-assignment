Essential Skills

HTML

HTML is short for Hypertext Markup Language, It is the standard markup language for documents designed to be displayed in a website.
It defines content and structure of web content. This is the first and basic skill to learn as a FullStack engineer.
It is the easiest skill to learn while starting out your journey as a FullStack engineer.
HTML is the first and basic tool to master as a FullStack Engineer, some might say it is the simplest to learn.
You can learn HTML on websites like CODEACADEMY, W3SCHOOLS, FREECODECAMP, UDEMY,
COURSERA, as well as on YOUTUBE which is the most common learning Tool.

CSS

CSS is short for Cascading Style Sheets, it is a style sheet language used for specifying
the presentation and styling of a document written in a markup language such as HTML.
CSS is used alongside HTML and JavaScript, it is the second skill that you have to learn on
your path to becoming a FullStack Engineer. You can learn CSS on websites like CODEACADEMY,
W3SCHOOLS, FREECODECAMP, UDEMY, COURSERA, as well as on YOUTUBE which is the most common learning Tool.

JavaScript

JavaScript is often abbreviated as JS. It is a programming languageand core technology of the Web, alongside with HTML and CSS.
Majority of all websites makes use of JavaScript on the client side for webpage behaviour.It is the engaging and aesthetic animations and effects you see on websites.
Web browsers have a dedicated JavaScript engine that executes the client code. You can learn JavaScript on websites like LEARN JAVASCRIPT, CODEACADEMY,
W3SCHOOLS, FREECODECAMP, UDEMY, COURSERA, as well as on YOUTUBE which is the most common learning Tool.

PYTHON

PYTHON is a high-level programming language. it is a general purpose programming language.
It's design philosophy emphasizes code readability with the use of significant indentation. Python is dynamically typed and
garbage-collected. It supports multiple programming paradigms, including structured, object-oriented and functional programming.
You can learn PYTHON from CODEACADEMY,
W3SCHOOLS, FREECODECAMP, UDEMY, COURSERA, as well as on YOUTUBE which is the most common learning Tool.

BACKEND TOOLS

BackEnd Tools to learn has a FullStack Engineer are numerous but i'll be mentioning a few of the most commonly used tools.
There is REACT JS, NODE JS, PYTHON, SQL, JAVASCRIPT, E.t.c. These tools are used to build the BackEnd part of the websites,
they ensure that the website is functional and can be used for the purpose it was made for. it is more complex than the Frontend part of website development.
You can learn these BACKEND Tools from websites like W3SCHOOLS, FREECODECAMP, UDEMY, COURSERA,
CODEACADEMY as well as on YOUTUBE which is the most common learning Tool.
